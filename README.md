# Meme Review

## System requeriments

- Ruby 2.5.1
- Ruby on Rails 6.0.0.beta3
- [Yarn](https://yarnpkg.com/en/docs/install#mac-stable)
- Node v8.10.0

## Get started

Execute

    yarn

And then execute:

    bundle install

Database:

    bin/rails db:create
    bin/rails db:migrate

Tests:

    bin/rails test

Rubocop:

    bundle exec rubocop

Start server:

    bin/rails s

    bin/webpack-dev-server
